# Install

```
npm install
```

# Initial Database

Create database
```
npx sequelize-cli db:create
```

Run database migration to create tables
```
npx sequelize-cli db:migrate
```

Seed data
```
npx sequelize-cli db:seed:all
```


# Run

```
npm start
```

Then open [http://localhost:3000](http://localhost:3000) in your browser.
